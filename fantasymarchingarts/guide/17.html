Events and Schedules

League founders are able to create events that are hosted by his or her league. This feature allows league members to automatically join league events on a first-come-first-served basis.


League Event Features

  • League money is used to create events
  • Tickets sales are given back to the league bank
  • Member can instantly join
  • A special Tier 7 championship event allows an unlimited number of league groups


Forming a League Schedule

Founders can add members' events to their league schedule, if desired. To add a member's event to the league schedule, simply go to the appropriate event page and click the appropriate link near the top. Once you add the event, it will show up in your league's schedule section.
