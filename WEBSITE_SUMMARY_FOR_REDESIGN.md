# Marching.Art - Comprehensive Website Summary for UI/UX Redesign

## Executive Overview

**Marching.Art** is an award-winning fantasy drum corps game that brings the excitement of DCI (Drum Corps International) to the digital world. Users create and manage virtual drum corps, select caption lineups from historical performances, and compete in real-time scoring during the DCI season.

**Tagline:** "The Ultimate Fantasy Drum Corps Game - Where Legends Are Made"

**Technology Stack:**
- Frontend: React 18.3, Vite, Tailwind CSS 3.4, Framer Motion 11
- State: Zustand + React Query + React Context
- Backend: Firebase (Auth, Firestore, Cloud Functions, Storage)
- Icons: Lucide React
- Hosting: Vercel (frontend) + Firebase (backend)
